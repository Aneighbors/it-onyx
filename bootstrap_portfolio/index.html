Programming Examples

Programming Example 1

This is a Mars Rover Database, that utilizes an AJAX call to nasa.api.gov and pulls over photos of the Mars rover missions. You can click on which rover you want Curiosity, Opportunity, or Spirit. Once you select a rover the picture date automatically sets to the first date of the mission, you can click search, or select a later date and search. It will then show you the pictures from that date as thumbnails, you can hover over each picture and a tooltip will appear with the name of the camera. You can also select the picture and a larger image will pop up.

Database Example 1

This database utilizes Excel and many of its functions such as Count, CountIFS, Sum, Average, if Statements, conditional Format and more. This is a fictional inventory for a Jewelry store.
